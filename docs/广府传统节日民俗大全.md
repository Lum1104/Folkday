# 广府传统节日民俗大全

> 广府地区指以广州为中心的珠三角粤语区，包括广州、佛山、顺德、番禺、南海、中山、东莞、珠海等地。
> 本文档涵盖广府地区全年主要传统节日、神明诞辰及民俗活动，供参考查阅。

---

## 一、春节系列（腊月廿三至正月十五）

广府人的春节从腊月廿三"谢灶"开始，到正月十五元宵节结束，前后近一个月。民谚云："年廿三，洗衣衫；年廿四，扫屋子；年廿七，执归一；年廿八，洗邋遢；年廿九，脏物都搬走；年卅晚，团年饭后去卖懒。"

### 1.1 年廿三 / 年廿四 —— 谢灶（祭灶神）

- **日期**: 农历腊月二十三或二十四（民间说法不一，"官三民四"）
- **背景来历**: 灶君（灶王爷）为一家之主，每年腊月廿三/廿四上天向玉帝汇报家中善恶。广府人称祭灶为"谢灶"，意为感谢灶君一年庇佑，并求其"上天言好事，下界保平安"。
- **需要做什么**:
  - 在灶台前摆放祭品，点红烛，上香拜三拜，敬酒敬茶
  - 祈求灶君上天言好事
  - 用糖果（尤其是麦芽糖/糖冬瓜）粘住灶君嘴巴，使其只说甜言蜜语
  - 焚烧灶君衣纸、金银衣纸，送灶君上天
- **准备物品**: 斋菜、烧肉、鸡、桔子、糖果（麦芽糖、糖冬瓜）、茶、酒、金银衣纸、香烛、灶君衣
- **重要程度**: **高**

### 1.2 年廿四至廿七 —— 开炸、蒸糕

- **日期**: 农历腊月二十四至二十七
- **背景来历**: 谢灶后即进入年前准备阶段。"廿四开炸"即开油锅炸年货，"廿五蒸糕"制作各式糕点。
- **需要做什么**:
  - **开油锅**: 炸煎堆（寓意"金银满屋"）、炸油角（寓意"荷包饱胀"）
  - **蒸年糕**: 蒸年糕（寓意"步步高升"）、萝卜糕、马蹄糕、芋头糕等
  - 办年货：购买瓜子、糖果、利是封、春联、桃花、年桔等
  - 年廿五至廿七陆续大扫除
- **准备物品**: 糯米粉、花生、芝麻、椰丝、萝卜、马蹄、红糖、油等原材料
- **重要程度**: **高**

### 1.3 年廿八 —— 洗邋遢

- **日期**: 农历腊月二十八
- **背景来历**: "年廿八，洗邋遢，若然不洗岂会发。"这是广府地区家喻户晓的年俗口诀。大扫除象征洗去旧年霉运、迎接新年好运。
- **需要做什么**:
  - 全屋大扫除：扫地、抹窗、洗厨房、清洁家什
  - 用**碌柚叶**（柚子叶）煲水洗澡，寓意洗去霉运、辟邪
  - 换上干净衣物和床品
  - 贴春联、挥春、门神、利是、"福"字
- **准备物品**: 碌柚叶、清洁用品、春联、挥春、门神画、"福"字贴
- **重要程度**: **高**

### 1.4 年廿八至年三十 —— 行花街（迎春花市）

- **日期**: 农历腊月二十八至除夕夜（连续三天三夜）
- **背景来历**: 广州花市可追溯到明代，芳村花地已是花木产区。新中国成立后，年宵花市固定为腊月廿八起连续三天。"未行过花街唔算过年"是广州人的共识。行花街已列入**国家级非物质文化遗产**。
- **需要做什么**:
  - 除夕吃完团年饭后全家逛花市（最热闹时段）
  - 选购年花：桃花（寓意大展宏图/桃花运）、水仙（寓意吉祥如意）、年桔（寓意大吉大利）、兰花、菊花、百合等
  - 选购年宵用品、风车、工艺品
  - 花市至年初一凌晨"收市"前竞价"执平花"也是一大特色
- **准备物品**: 现金/支付工具（买花）
- **重要程度**: **高**（广府年俗最核心活动之一）

### 1.5 年三十（除夕）—— 团年饭、守岁、卖懒

- **日期**: 农历腊月三十（小月则二十九）
- **背景来历**: 除夕是一年最后一天，广府人极为重视"团年"（团圆）。全家人必须齐聚吃团年饭。
- **需要做什么**:
  - **祭祖**: 下午在家中神位/祖先牌位前摆设祭品，上香拜祭
  - **团年饭**: 全家围坐吃年夜饭（菜式必有鸡、鱼、发菜蚝豉/发财好市、猪手/横财就手、生菜/生财等意头菜）
  - **压岁钱/利是**: 长辈给晚辈"压岁"
  - **守岁**: 除夕夜不睡觉，等待新年到来
  - **卖懒**: 旧时广府小孩子除夕夜提灯笼沿街叫卖"卖懒！卖懒！卖到年三十晚！"寓意卖掉懒惰，新年勤奋
  - **行花街**: 团年饭后全家出动逛花市
- **准备物品**: 鸡、鱼、发菜、蚝豉、猪手、年糕、汤圆、利是封、香烛、元宝蜡烛
- **重要程度**: **高**

### 1.6 正月初一 —— 拜年

- **日期**: 农历正月初一
- **背景来历**: 新年伊始，广府人有"拜年"风俗。初一通常拜本家长辈。
- **需要做什么**:
  - 穿新衣、戴新帽
  - 早上吃**斋菜**（广州不少家庭初一吃素，求一年清净平安）
  - 给长辈拜年，说"恭喜发财"
  - **派利是/逗利是**: 已婚者向未婚者派利是（红包），金额不论多少图个意头
  - 初一不扫地、不倒垃圾（怕扫走财气）
  - 不说不吉利的话
  - 放鞭炮（部分地区）、舞醒狮
- **准备物品**: 新衣服、利是封、桔子（拜年手信）、糖果瓜子（待客）
- **重要程度**: **高**

### 1.7 正月初二 —— 开年

- **日期**: 农历正月初二
- **背景来历**: 初二称"开年"，年节正式启动。广州人时兴初二、十六"打牙祭"，正月初二为一年之中的"头牙"。
- **需要做什么**:
  - **开年饭**: 一大早"开鍋"，下花生油，煎萝卜糕、煎鱼（"有余"）、煎年糕
  - 吃开年饭（必有鸡、鲤鱼、生菜、萝卜糕等）
  - 回娘家拜年（已嫁女儿携夫婿回娘家，广府称"返外家"）
  - 拜土地公（头牙）
  - **开市**: 商铺择吉时开门做生意，放鞭炮、舞醒狮
- **准备物品**: 萝卜糕、年糕、鸡、鲤鱼、利是、生菜、果品
- **重要程度**: **高**

### 1.8 正月初三 —— 赤口（小年朝）

- **日期**: 农历正月初三
- **背景来历**: 广府人称初三为"赤口"，认为这天容易与人争吵，不宜拜年。
- **需要做什么**:
  - 一般不出门拜年
  - 在家休息或去庙宇祈福
  - "赤口"贴：将写有"赤口"二字的红纸条贴在门口或扔到路口
- **准备物品**: 赤口帖（红纸写"赤口"）
- **重要程度**: **中**

### 1.9 正月初四 —— 接财神

- **日期**: 农历正月初四
- **背景来历**: 传说初四是诸神回到人间的日子，各家各户要迎接灶神和各路神仙回来。
- **需要做什么**:
  - 迎灶神回家
  - 备果品、茶酒接神
  - 放鞭炮迎财神
- **准备物品**: 果品、茶酒、香烛、鞭炮
- **重要程度**: **中**

### 1.10 正月初五 —— 破五/接财神

- **日期**: 农历正月初五
- **背景来历**: 初五俗称"破五"，之前的各种禁忌至此解除。也是迎财神的重要日子。
- **需要做什么**:
  - 可以打扫卫生（倒垃圾）
  - 商铺正式开市营业
  - 迎财神
- **准备物品**: 香烛、鞭炮
- **重要程度**: **中**

### 1.11 正月初七 —— 人日（众人生日）

- **日期**: 农历正月初七
- **背景来历**: 源自汉代传说，女娲造人于正月初七。广府人俗称"众人生日"，南北朝《荆楚岁时记》已有记载。
- **需要做什么**:
  - **吃七样菜**（又称"七宝羹"）: 将七种新鲜蔬菜同煮，是人日核心习俗
  - 七样菜中五种固定：芹菜（勤快）、芥菜（发财）、菠菜（长命）、青葱（聪明）、大蒜（会算/精打细算），另外两种因地区而异，常见有生菜（生财）、芫荽（缘分）等
  - 设丰盛家庭筵席，庆贺"人的生日"
- **准备物品**: 芹菜、芥菜、菠菜、青葱、大蒜、生菜、芫荽等七种蔬菜
- **重要程度**: **高**

### 1.12 正月初九 —— 天公诞

- **日期**: 农历正月初九
- **背景来历**: 正月初九为玉皇大帝（天公）诞辰。广府地区部分家庭会拜天公。
- **需要做什么**:
  - 凌晨子时或初九清晨拜天公
  - 备丰盛祭品敬拜
- **准备物品**: 三牲、果品、茶酒、香烛、金银衣纸
- **重要程度**: **中**

### 1.13 正月十五 —— 元宵节

- **日期**: 农历正月十五
- **背景来历**: 元宵节又称"上元节"，是春节系列的收官之作。广府人闹元宵热闹非凡。
- **需要做什么**:
  - **吃汤圆**（寓意团圆美满）
  - **赏花灯/猜灯谜**
  - **广府庙会**（详见下文）
  - **佛山行通济**（详见下文）
  - 舞醒狮、舞龙
- **准备物品**: 汤圆、花灯
- **重要程度**: **高**

## 二、佛山行通济

- **名称**: 行通济
- **日期**: 农历正月十四至正月十六（以正月十六为正日）
- **地点**: 佛山市禅城区通济桥
- **背景来历**: 通济桥始建于明代，是古时佛山通往顺德、番禺的商贸要道。商民为求生意顺遂而行桥祈福，逐渐形成"正月十六行通济"的民俗，距今已有400余年历史。俗语"行通济，冇闭翳"（行通济，无忧愁）家喻户晓。已列入**广东省非物质文化遗产**。
- **需要做什么**:
  - 携家带口步行走过通济桥
  - 手持**风车**（转运）和**生菜**（生财）
  - 过桥后将生菜抛入河中或投入慈善捐款处
  - 不走回头路（寓意一路畅通）
  - 沿途观赏龙狮表演、花灯
- **准备物品**: 风车、生菜（谐音"生财"）
- **重要程度**: **高**（佛山年度最大规模民俗活动，每年参与人数超百万）

## 三、广府庙会

- **名称**: 广府庙会
- **日期**: 农历正月十五（元宵节）起，连续七天（至正月廿一）
- **地点**: 广州市越秀区，以城隍庙忠佑广场为主会场，北京路文化核心区为中心辐射周边
- **背景来历**: 广府庙会源于明清城隍庙祭祀民俗，曾一度中断。2011年越秀区牵头恢复，至今已连续举办十余届。围绕"城脉、文脉、商脉"三大核心脉络展开，是广州元宵节期间最大型的综合性文化活动。
- **需要做什么**:
  - 逛庙会（免费开放）
  - 观赏非遗巡游、醒狮表演、粤剧曲艺
  - 参与猜灯谜、民俗手工体验
  - 品尝广府小吃美食
  - 逛国潮市集、文创产品
  - 广府达人秀等文艺展演
- **准备物品**: 无特殊准备
- **重要程度**: **高**（广州元宵节最大民俗品牌活动，每年吸引数百万人参与）

## 四、波罗诞（南海神诞）

- **名称**: 波罗诞（又称南海神诞、洪圣诞）
- **日期**: 农历二月十一至二月十五（以二月十三正诞日为核心）
- **地点**: 广州市黄埔区穗东街道南海神庙（波罗庙）
- **背景来历**: 波罗诞是为庆祝中国四大海神之一的南海神（祝融）诞辰而举行的迎神赛会，已有一千余年历史，是广州最古老、最盛大、最具影响力的千年庙会。"波罗"之名源于南海神庙旁的波罗树。已列入**国家级非物质文化遗产**。
- **需要做什么**:
  - 前往南海神庙祭拜南海神
  - 购买"波罗鸡"（用棉纸和谷壳制作的工艺品，波罗诞标志性纪念品）
  - 观赏非遗展演、民俗巡游、粤剧表演
  - 品尝传统祭祀菜式（十五乡乡民设宴请亲朋）
  - 参观广州海事博物馆
- **准备物品**: 香烛、祭品（可在庙前购买）
- **重要程度**: **高**（广州十大非遗节庆活动之一）

## 五、清明节（行清/拜山）

- **名称**: 清明节（广府人称"行清"或"拜山"）
- **日期**: 公历4月4日或5日（清明节气）；广府人讲究在"正清"（清明当日）"行清"
- **背景来历**: 广府人非常重视清明祭祖。粤语"行清"指全族人约定时间一起去扫墓祭祖，"行正清"专指清明当天扫墓。有的地方要求新坟头三年必须"行正清"，三年后可在清明后一个月内完成祭拜。
- **需要做什么**:
  - **拜山（扫墓）**: 全家或全族人齐齐整整去拜山
  - 清除祖墓周围杂草，为坟墓培上新土
  - 折嫩绿新枝插在坟上
  - 摆放祭品、纸钱，上香、敬酒、拜祭
  - 焚化"财帛"（金银衣纸）
  - 吃供品（祭毕分食祭品）
  - **插柳**: 在祖宗牌位和门口插"鬼怖木"（柳枝）以避野鬼
  - 买风车回家（纸蛇风车，代表"坟龙"保佑消灾得福）
  - **吃清明蔗**：拜山祭祖的"指定动作"
  - 吃清明虾（时令）
- **准备物品**:
  - 祭品：烧猪/金猪、鸡、鲤鱼、发糕、甘蔗、苹果、酒
  - 纸钱、金银衣纸、香烛
  - 锄头、镰刀（除草培土用）
  - 柳枝
- **重要程度**: **高**（广府人极为重视的祭祖大节）

## 六、端午节（五月节/扒龙舟）

- **名称**: 端午节（广府人俗称"五月节"）
- **日期**: 农历五月初一至五月十八（以五月初五为正日，但龙舟活动贯穿大半个月）
- **背景来历**: 广州端午扒龙舟的传统由来已久，明清时期已十分普遍。广州扒龙舟已列入**国家级非物质文化遗产**。端午龙舟活动不只是比赛，更是一套完整的宗族文化仪式。

### 6.1 龙舟全套流程

#### 起龙（农历四月初八前后）
- 将去年端午后沉入涌底的龙船起出
- 清淤泥、检查船身、修补、上油、上漆
- 举行起龙仪式，焚香拜神

#### 采青
- 龙船起出修好后，到河边或田野采集青草扎在船头
- 寓意生机勃勃、龙船精神

#### 招景/趁景（五月初一至十八）
- **趁景**: 各村约定日期，扒龙船互访联谊，今天在这村，明天在那村
- **招景**: 本村设宴招待来访龙船
- 只表演不比赛，重在联络乡谊
- "龙舟景"是广府水乡最具特色的民俗活动

#### 赛龙/斗标（五月初五前后）
- 正式比赛竞渡，俗称"扒龙舟"
- 锣鼓震天，龙船竞速
- 各村拼尽全力争夺锦标

#### 龙船饭
- 赛龙或趁景后，全村在祠堂或空地大排筵席
- 邀请亲朋好友共食龙船饭
- 菜式丰盛，十大碗起步（烧肉、白切鸡、焖猪肉、蒸鱼等）
- 吃龙船饭可得好兆头

#### 藏龙/散龙
- 端午活动结束后，将龙船重新沉入河涌底保存
- 举行散标仪式
- 来年再起龙

### 6.2 端午其他习俗

- **食粽**: 广府粽以咸肉粽为主（糯米、五花肉、绿豆、咸蛋黄）
- **饮午时茶**: 用特定中草药煲凉茶，在午时饮用
- **挂艾草菖蒲**: 门口悬挂辟邪
- **佩香囊**: 小孩佩戴装有中药的香囊
- **龙船饼**: 龙船饭配套的传统点心
- **吃龙船糕**: 糯米粉红豆沙制成龙船形糕点

- **准备物品**: 粽子原料、艾草、菖蒲、香囊、凉茶草药
- **重要程度**: **高**（广府人最隆重的夏季节日）

## 七、七夕/乞巧节（七姐诞）

- **名称**: 七夕节/乞巧节（广府人称"七姐诞"或"七娘诞"）
- **日期**: 农历七月初六至初七（初六摆巧，初七正诞）
- **地点**: 广州天河区珠村为"中国乞巧第一村"
- **背景来历**: 广州乞巧节习俗历史悠久，与中原地区的区别在于广府乞巧重在"摆"（展示手工艺品）而非"拜"。珠村乞巧传统尤为突出。2005年起，首届广州乞巧文化节在珠村开幕。广州乞巧已列入**国家级非物质文化遗产**。
- **需要做什么**:
  - **摆七娘（摆巧）**: 未婚女子在家门前或厅堂摆设八仙桌，展示自己制作的手工艺品（刺绣、剪纸、微型家具等），配以古董珍玩、鲜花时果、胭脂红粉
  - "大七娘"需多张八仙桌，制作精美，展示7天；"小七娘"规模较小
  - **拜七娘**: 初七夜晚对着明月，在供案前诚心拜祭，乞求心灵手巧
  - **赛巧/卜巧**: 进行乞巧游戏，如穿针引线比赛
  - **睇七娘**: 街坊邻里相互欣赏各家的"摆巧"作品
  - **送七娘**: 节日结束焚化纸制祭品，送别七姐
- **准备物品**: 手工艺品（刺绣品、剪纸、微型家具）、鲜花、时令水果、胭脂、香粉、针线、香烛
- **重要程度**: **高**（广州十大非遗节庆活动之一）

## 八、盂兰节/中元节（烧街衣）

- **名称**: 盂兰节/中元节（广府人俗称"鬼节"、"七月半"、"烧衣"）
- **日期**: 农历七月十四（广府人以七月十四为正日，而非北方的七月十五）
- **背景来历**: 道教将七月十五视为中元节（地官赦罪之日），佛教称盂兰盆节。广府人以七月十四为主要祭祀日，俗称"烧衣"。农历七月整月被视为"鬼月"，阴间之门打开，亡魂游荡人间。
- **需要做什么**:
  - **祭祖**: 在家中设供桌，摆放祭品拜祭祖先
  - **烧街衣**: 入夜后在路边、树下焚烧纸衣、金银衣纸，供无主孤魂御寒果腹
  - 供奉食物（白饭、豆腐、芽菜、水果等）给游魂
  - 焚香、撒纸钱
  - 广府习俗中拜地官，为人赦罪
  - 部分地区有盂兰胜会、神功戏
- **准备物品**: 纸衣、金银衣纸、纸元宝、香烛、白饭、豆腐、水果等祭品
- **禁忌**:
  - 七月期间避免夜间外出
  - 不随意拍他人肩膀
  - 避免在路边踩踏祭品
- **重要程度**: **高**

## 九、中秋节

- **名称**: 中秋节
- **日期**: 农历八月十五
- **背景来历**: 中秋赏月、吃月饼的习俗全国皆有，但广府地区有多种独特的庆祝方式，如拜月光、竖中秋、舞火龙等。

### 9.1 拜月光（拜月娘）

- 入夜后，由家中德高望重的长辈摆开供桌
- 摆上月饼、田螺、芋头、菱角、柚子、杨桃等贡品
- 茶酒在旁，焚香礼拜
- 礼毕全家围坐赏月
- 小孩提灯笼玩耍

### 9.2 竖中秋（树中秋）

- "树"亦作"竖"，即将灯彩高竖起来
- 用竹条扎灯笼，鱼龙造型多样
- 小孩在家长协助下扎灯笼，挂在屋顶或天井
- 互相比赛谁竖得高、竖得多、灯彩最精巧

### 9.3 舞火龙

- 广东佛山地区及广州白云区、花都区特有的中秋民俗
- 用稻草扎成龙形，插上燃着的香
- 由青壮年赤膊上阵，挥舞舞动
- 传说舞火龙可趋吉避凶、风调雨顺
- 佛山上元村的舞火龙传承已有500年
- 广州白云区江高镇等地也有舞火龙传统

### 9.4 烧禾楼

- 花都炭步等地特有中秋习俗
- 用稻草、砖块搭建"禾楼"点火焚烧

### 9.5 其他中秋习俗

- **吃月饼**: 广式月饼（莲蓉、豆沙、五仁、蛋黄等）
- **吃田螺**: 中秋时节田螺最肥美
- **吃芋头**: 寓意辟邪消灾
- **吃菱角**: 时令应节
- **食柚子**: 柚与"佑"谐音

- **准备物品**: 月饼、田螺、芋头、菱角、柚子、杨桃、灯笼、香烛
- **重要程度**: **高**

## 十、重阳节（登高）

- **名称**: 重阳节（广府人称"重九"）
- **日期**: 农历九月初九
- **背景来历**: 广州重阳登高习俗源自中原，明清时期已盛行于越秀山、双塔等地。2007年"广州重阳登高"列入广州市第一批**市级非物质文化遗产**。
- **需要做什么**:
  - **登高**: 全城市民涌向白云山、越秀山、帽峰山等地登高，广州塔等新地标也成为新选择
  - **打地气**: 广东特有说法，重阳前一天提前上山沾地气、感受自然气息，在山上静思过夜迎接朝阳
  - **祭祖**: 重阳也是重要的祭祖日
  - **放纸鸢（风筝）**: 岭南地区重阳特色，佛山地区尤为盛行，纸鸢上系藤弓发出响声（"响弓鹞"）
  - **赏菊**: 插菊花、饮菊花酒/菊花茶
  - **佩茱萸**: 驱邪求吉
  - **吃重阳糕**: 寓意"步步高升"
  - **敬老**: 现代重阳也是敬老节
- **准备物品**: 风筝、菊花、茱萸、重阳糕、登山装备
- **重要程度**: **高**（广州重阳登高为非遗项目，全城参与）

## 十一、冬至

- **名称**: 冬至（广府人称"冬节"、"过冬"）
- **日期**: 公历12月21日或22日（冬至节气）
- **背景来历**: 广府人有句代代相传的俗语——**"冬至大过年"**（冬至比过年还重要），可见冬至在广府人心中的地位。古人视冬至为一年最重要的节日之一。1956年《广州日报》曾记载旧时广州人过冬至的"规则"：拜祖先、饲耗、食汤圆、做腊味糯米饭、宰鸡杀鸭炖八珍。
- **需要做什么**:
  - **祭拜祖先**: 杀鸡劏鸭，准备汤圆、果品等祭拜祖先。有的地方杀猪祭祖，在祠堂按人丁分猪肉
  - **冬祭**: 佛山南海等地宗族祠堂举行隆重冬祭，长者穿长衫马褂领族人行三跪九叩礼、诵祭文
  - **吃团圆饭（"团冬"）**: 全家人围坐吃丰盛冬至饭，必有鸡、鸭、烧肉
  - **食汤圆**: 甜汤圆象征团圆美满（广州汤圆通常为饭后甜汤）
  - **做腊味糯米饭**: 冬菇、虾米、瑶柱、腊肠等丰富配料
  - **炖补汤**: 冬至进补，炖鸡汤、药材汤
  - **饲耗**: 旧俗，冬至夜将汤圆粘贴在家中各处（门框、灶台、米缸等），"喂老鼠"以求来年不受鼠害
  - **喂耕牛**: 旧时农家冬至让耕牛吃好料犒劳
- **准备物品**: 鸡、鸭、烧肉、汤圆（糯米粉）、腊味糯米饭材料、补汤药材、香烛、果品
- **重要程度**: **高**（"冬至大过年"，广府人极为重视的节日）

## 十二、神明诞辰

广府地区民间信仰丰富，一年中各种"神诞"络绎不绝，以下列出主要神诞：

### 12.1 观音诞

- **日期**:
  - 农历二月十九（观音诞生日/观音开库日）
  - 农历六月十九（观音得道日）
  - 农历九月十九（观音出家日）
  - 农历正月初八（白衣观音诞，部分地区）
- **背景来历**: 观音菩萨是广府地区最普遍、最广泛信仰的神明之一。"观音开库"（正月廿六或二月十九）尤为热门——传说观音菩萨在此日打开宝库借钱给世人，信众可前往"借库"求财。
- **需要做什么**:
  - 前往观音庙（广州大佛寺、光孝寺、六榕寺等）上香祈福
  - **观音开库**: 排队"借库"，求一年财运
  - 供奉鲜花、水果、素斋
  - 吃斋（观音诞吃素是传统）
- **准备物品**: 鲜花、水果、素斋供品、香烛
- **重要程度**: **高**

### 12.2 北帝诞（玄天上帝诞）

- **日期**: 农历三月初三
- **背景来历**: 北帝即玄天上帝（真武大帝），是广府地区重要的水神和保护神。佛山祖庙即为北帝庙，北帝信仰在广府地区极为普遍。庆祝活动始于明朝万历年间，距今近500年。
- **需要做什么**:
  - 前往北帝庙（佛山祖庙、广州仁威庙等）祭拜
  - 盛大巡游：抬北帝神像出游
  - 醒狮表演、粤剧大戏
  - 斋宴
  - 番禺沙湾地区同日举行飘色巡游
- **准备物品**: 三牲、香烛、金银衣纸
- **重要程度**: **高**（佛山祖庙北帝诞为广府地区最重要的神诞之一）

### 12.3 天后诞/妈祖诞

- **日期**: 农历三月廿三
- **背景来历**: 天后（妈祖）是海上守护神，广府沿海和水上居民极为敬奉。广州南沙天后宫是珠三角最大的妈祖庙。南沙妈祖信俗已列入广州十大非遗节庆活动。
- **需要做什么**:
  - 前往天后庙祭拜（南沙天后宫、黄埔天后宫等）
  - 张灯结彩、打醮庆贺
  - 迎神赛会、神像巡游
  - 粤剧神功戏
  - 传说诞会前会捉一只大蝴蝶回宫，让它爬上墙，示"天后娘娘回宫"
- **准备物品**: 三牲、鲜花、香烛、金银衣纸
- **重要程度**: **高**（尤其是沿海地区和水上人家）

### 12.4 洪圣诞（洪圣爷诞）

- **日期**: 农历二月十三（与波罗诞同日/同源）
- **背景来历**: 洪圣即南海神（祝融），广府人称"洪圣爷"。洪圣诞与波罗诞同源，各地洪圣庙均有庆祝。
- **需要做什么**:
  - 前往洪圣庙祭拜
  - 醒狮、巡游、粤剧表演
- **准备物品**: 三牲、香烛
- **重要程度**: **中**

### 12.5 华光诞（华光大帝诞）

- **日期**: 农历九月廿八
- **背景来历**: 华光大帝（马天君/五显灵官大帝）是粤剧行业的祖师爷和保护神，也被视为火神。粤剧界和广府娱乐行业极为敬奉。
- **需要做什么**:
  - 粤剧行业举行祭祀活动
  - 演出"华光大帝"题材粤剧
  - 上香祈福
- **准备物品**: 三牲、果品、香烛
- **重要程度**: **中**（粤剧界和演艺界非常重视）

### 12.6 龙母诞

- **日期**: 农历五月初八（诞辰）、农历八月十五（另一诞期，部分地区）
- **背景来历**: 龙母是西江流域的水神和保护神，传说她养育五条小龙。龙母信仰盛行于广东西部和广西，珠三角地区也有龙母庙。
- **需要做什么**:
  - 前往龙母庙祭拜（德庆龙母祖庙为正宗，广州等地也有龙母庙）
  - 放生鲤鱼（龙母诞最壮观的活动）
  - 吃龙船糕
  - 许愿祈福
  - 供奉香盒（龙母赐福图案）
- **准备物品**: 鲤鱼（放生用）、香烛、果品、龙船糕
- **重要程度**: **中**

### 12.7 金花夫人诞（金花诞）

- **日期**: 农历四月十七
- **背景来历**: 金花夫人（金花娘娘/金花圣母）是广府人传说中的生育女神和妇女儿童保护神。据《广州府志》记载，金花夫人原为广州巫女，明朝洪武年间因助巡抚陈濂夫人顺产而得名。金花娘娘诞已列入广东省非物质文化遗产。
- **需要做什么**:
  - 前往金花庙祭拜
  - 求子求福，保佑儿童平安
  - 供奉花、果
- **准备物品**: 鲜花、水果、香烛
- **重要程度**: **中**（求子家庭和有幼儿的家庭尤为重视）

### 12.8 何仙姑诞

- **日期**: 农历三月初七
- **背景来历**: 何仙姑为八仙之一，传说为广州增城人。增城何仙姑庙为主要祭祀场所。
- **需要做什么**:
  - 前往何仙姑庙（增城小楼镇）祈福
  - 参观何仙姑家庙
- **准备物品**: 香烛、果品
- **重要程度**: **低**

### 12.9 关帝诞（关公磨刀日）

- **日期**: 农历五月十三
- **背景来历**: 关羽（关公/关帝）被历代奉为忠义楷模，广府商界尤为敬奉（关公也是武财神）。
- **需要做什么**:
  - 商铺、会馆祭拜关帝
  - 上香祈求生意兴隆
- **准备物品**: 三牲、香烛
- **重要程度**: **中**（商界人士重视）

### 12.10 土地诞（土地公诞）

- **日期**: 农历二月初二
- **背景来历**: 二月初二也是"头牙"，即一年中第一个"做牙"日。土地公（福德正神）是广府社区最基层的守护神。
- **需要做什么**:
  - 社区土地庙/社坛前祭拜
  - 社区联欢
  - "做牙"（详见下文"做牙"章节）
- **准备物品**: 三牲、香烛、果品
- **重要程度**: **中**

### 12.11 鲁班诞（鲁班先师诞）

- **日期**: 农历六月十三
- **背景来历**: 鲁班是建筑业和木工业的祖师爷，广府建筑行业传统上在此日祭祀。
- **需要做什么**:
  - 建筑业、木工业举行行业祭祀
- **准备物品**: 三牲、果品、香烛
- **重要程度**: **低**（行业性节日）

### 12.12 文昌诞

- **日期**: 农历二月初三
- **背景来历**: 文昌帝君是掌管功名学业的神明，读书人和求学之家敬奉。
- **需要做什么**:
  - 学子前往文昌庙祈求学业进步
  - 上香拜文昌
- **准备物品**: 葱（聪明）、芹（勤奋）、蒜（会算）、笔墨纸砚（象征意义）、香烛
- **重要程度**: **中**（学生家庭重视）

## 十三、做牙（每月初二、十六拜土地公）

- **名称**: 做牙（又称"打牙祭"）
- **日期**: 每月农历初二和十六
- **特殊做牙日**:
  - **头牙**: 农历二月初二（一年第一个做牙日，与土地诞同日）
  - **半年牙**: 农历六月十六
  - **尾牙**: 农历十二月十六（一年最后一个做牙日，最为隆重）
- **背景来历**: "做牙"起源于古代军中祭旗（牙旗）之礼，后来传到民间演变为商家定期祭拜土地公的习俗。广府地区商铺、工商业主每月初二、十六都会做牙，祈求土地公保佑生意兴隆。"打牙祭"一词亦由此而来。
- **需要做什么**:
  - 在店铺或家中门前摆设祭品拜土地公（福德正神）
  - 以三牲（白斩鸡、猪肉、鱼肉）祭拜
  - 祭后共享祭品，即"打牙祭"
  - **头牙**: 要"搶头牙"，一大早做好祭品抢先拜
  - **尾牙**: 最为隆重，老板宴请员工（尾牙宴），感谢员工一年辛劳，现代演变为"年会"
- **准备物品**: 三牲（鸡、猪肉、鱼）、香烛、酒、水果
- **重要程度**: **中**（头牙和尾牙为高，平时做牙为中/低）

## 十四、其他广府特有民俗

### 14.1 卖懒

- **日期**: 除夕夜
- **背景来历**: 广府独特的除夕夜民俗，源于中原"卖痴呆"习俗。
- **需要做什么**:
  - 小孩子手提灯笼，沿街叫喊："卖懒！卖懒！卖到年三十晚！人懒我唔懒！"
  - 每人带一个红鸡蛋和一炷香
  - 走到村口或街头将懒惰"卖"出
- **准备物品**: 灯笼、红鸡蛋、香
- **重要程度**: **中**（传统渐趋式微，但部分地区仍有保留）

### 14.2 舞醒狮

- **日期**: 春节期间（正月初一至十五）、开业庆典、各类喜庆场合
- **背景来历**: 广东醒狮是融武术、舞蹈、音乐为一体的传统文化活动，已列入**国家级非物质文化遗产**。以佛山、遂溪、广州为主要分布地。"醒"字取"醒目"之意。
- **需要做什么**:
  - 两人扮演一头狮子，一人头戴"大头佛"手执葵扇引狮
  - 锣鼓擂响，舞狮人展示各种高难度动作
  - **采青**: 舞狮核心环节，狮子采食挂在高处的"青"（系有利是的生菜），寓意生财
  - 高桩狮、地狮等不同流派
- **准备物品**: 醒狮头、锣鼓、生菜（采青用）、利是
- **重要程度**: **高**（广府最具代表性的民俗表演）

### 14.3 飘色（沙湾飘色）

- **日期**: 农历三月初三（北帝诞）前后为传统飘色巡游日；现多在重大节庆进行
- **地点**: 广州番禺沙湾为飘色之乡
- **背景来历**: 沙湾飘色被誉为"凝固的戏剧，活动的雕塑"，是广府民间艺术瑰宝。集材料、力学、音乐、造型、装饰等工艺于一体。
- **需要做什么**:
  - 观赏飘色巡游（小孩打扮成戏剧人物，站在高高的色板上，看似凌空飘浮）
  - 配合醒狮、鳌鱼、舞龙、鱼灯等表演
- **重要程度**: **高**（广州十大非遗节庆活动之一）

### 14.4 广府饮茶（叹早茶）

- **日期**: 全年每日（非节令性，但为广府最核心文化习俗）
- **背景来历**: "饮茶"即叹早茶，"一盅两件"是广府人的生活方式。广府饮茶习俗已列入市级非遗。
- **需要做什么**:
  - 到茶楼饮茶、吃点心（虾饺、烧卖、叉烧包、肠粉、凤爪等）
  - 与家人朋友社交叙旧
  - "揭盖续水"等茶楼礼仪
- **重要程度**: **高**（广府人日常核心文化）

### 14.5 派利是（红包文化）

- **日期**: 春节期间及开工日
- **背景来历**: 广府利是文化与北方红包不同——已婚者不论长幼一律要派利是，未婚者不论长幼一律可逗利是。金额通常不大（5元、10元、20元常见），重在意头而非金额。
- **需要做什么**:
  - 已婚人士准备利是封，见面派发
  - 未婚人士主动"逗利是"（讨红包）
  - 节后开工日，公司派"开工利是"
- **准备物品**: 利是封（红包袋）、新钞票
- **重要程度**: **高**

### 14.6 猎德新春巡游

- **日期**: 农历正月初十前后（近年为固定日期）
- **地点**: 广州天河区猎德村
- **背景来历**: 猎德社区新春巡游有百年历史，2024年复办后在天河CBD举行，成为城中村与现代都市融合的民俗亮点。
- **需要做什么**:
  - 观赏醒狮、舞龙巡游
  - 感受广府宗族文化
- **重要程度**: **中**
